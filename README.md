# Naavya Portfolio

A modern luxury portfolio website built using React, TypeScript, Vite and Tailwind CSS.

## Features

- Responsive Design
- Interactive Spotlight Hero Section
- Projects Showcase
- Skills Section
- Certifications Section
- Contact Section
- Resume Download

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
