🔗 https://naavya-portfolio-ueb4.vercel.app/

- Luxury Dark UI
- Fully Responsive Design
- Interactive Hero Section
- Spotlight Reveal Effect
- About Section
- Skills Showcase
- Project Gallery
- Contact Section
- Resume Download
