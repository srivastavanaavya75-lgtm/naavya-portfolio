# Naavya Srivastava Portfolio

A premium personal portfolio built using React, TypeScript, Vite and Tailwind CSS.

[Vercel Link]

## Features

- Luxury dark UI
- Responsive Design
- Interactive Hero Section
- Spotlight Reveal Effect
- About Section
- Skills Showcase
- Project Gallery
- Contact Section
- Resume Download

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Installation

git clone <repo>

cd naavya-portfolio

npm install

npm run dev
